Use v-for directive for dealing with iterable objects in the associated Vue instance

example-7

v-model is a directive that works on input-receiving HTML tags. So, tags such as `input`, `select`, `textarea` that usually lie within a `form` tag

We also look at `methods` property that accepts an object literal whose properties are functions that can modify the object backing the Vue-instantiated

Do not forget to use `.prevent` modifier to modify the `v-on:submit` directive. Because normally, the submit button within a form, upon clicking, will lead to another URL. We are not POST-ing the form in this example

This is the list of members available. The list is contained in `memberNames`, which is a property of the object associated with `data` property of the Vue instance
